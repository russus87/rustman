Funzionalità

Tutto quello che ti serve per le tue API

Le stesse comodità di Postman, in un'app nativa che parte in un istante.

📁

Collezioni & cartelle

Organizza le richieste in cartelle annidate, salvate come file JSON facili da versionare.

🔀

Git integrato

Commit, diff, cronologia e remote con pull/push direttamente sul tuo workspace.

✅

Test & asserzioni

Verifica le risposte e lancia performance test visualizzando i risultati con grafici.

🌐

Environments

Variabili {{...}}, query params e auth Bearer/Basic per ogni ambiente.

📜

Pre/Post-script

Scrivi script JavaScript con API in stile Postman (pm.*) prima e dopo le chiamate.

⛓️

Run & integration

Concatena chiamate per gli integration test, con import/export e workspace multipli.
